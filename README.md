Welcome to the **Data Warehouse and Analytics Project** repository! 🚀

Designed as a **portfolio project**, it highlights industry best practices in **data engineering, modeling, and analytics**.  

---

## 🏗️ Data Architecture

The project follows the **Medallion Architecture** with **Bronze**, **Silver**, and **Gold** layers:

<p align="center">
  <img src="docs/data_architecture.png" alt="Data Architecture" width="600"/>
</p>

- **🥉 Bronze Layer** → Raw data as-is from source systems (CSV → SQL Server).  
- **🥈 Silver Layer** → Data cleansing, standardization, and normalization.  
- **🥇 Gold Layer** → Business-ready star schema for reporting & analytics.  

---

## 📖 Project Overview

This project involves:

1. **Data Architecture** – Medallion design with Bronze/Silver/Gold layers.  
2. **ETL Pipelines** – Extract, transform, load from sources → warehouse.  
3. **Data Modeling** – Fact & dimension tables for analytics.  
4. **Analytics & Reporting** – SQL dashboards & insights.  

🎯 Great for showcasing expertise in:  
`SQL Development • Data Architecture • Data Engineering • ETL Pipelines • Data Modeling • Data Analytics`

## 🚀 Project Requirements

### 🔹 Data Engineering – Building the Warehouse
**Objective**  
Develop a **modern data warehouse** in SQL Server to consolidate sales data for analytics.  

**Specifications**  
- **Sources**: ERP & CRM CSV files.  
- **Data Quality**: Cleansing & validation.  
- **Integration**: Merge into a unified analytical model.  
- **Scope**: Latest dataset only (no historization).  
- **Docs**: Data model documentation for stakeholders & analysts.  

---

### 🔹 Data Analysis – BI & Reporting
**Objective**  
Deliver SQL-based analytics on:  
- 👥 Customer Behavior  
- 📦 Product Performance  
- 📈 Sales Trends
